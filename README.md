# Scroball
Universal Last.fm scrobbler for Android

Scroball is a lightweight scrobbler for Last.fm which can track music playback from almost any media player on Android, including:

- Apple Music
- Pandora
- Google Play Music
- Spotify
- SoundCloud
- YouTube Red
- Deezer
- Tidal
- Amazon Music
- And many more...
